# タスク完了時チェックリスト

## 必須実行項目（AI開発効率化ルール）

### 1. 品質チェック必須実行
実装完了時に以下を**必ず実行**し、エラー0件を確認：

```bash
# TypeScriptエラー0件が必須（最優先）
cd layers/base && bun typecheck

# Lintエラー0件が必須
cd layers/base && bun lint

# 関連テスト全て成功が必須
cd layers/base && bun test:ut
```

### 2. 自動停止トリガー
以下の条件で即座に作業停止し、ユーザーに報告：
- **TypeScriptエラー残存**: 1件でも残存時は即座停止
- **大規模変更**: 3ファイル以上の同時変更時は影響範囲レポート作成
- **繰り返しエラー**: 同じエラー3回発生時は根本原因分析必須
- **デバッグコード残存**: console.log等の削除確認
- **any型使用**: 型安全性違反として即座停止

### 3. 品質エラー対処優先順位
エラー発見時の対処順序：
1. **TypeScriptエラー**: 即座に修正（最優先）
2. **Lintエラー**: 即座に修正
3. **テスト失敗**: 実装停止して原因分析・修正
4. **ビルドエラー**: 根本原因分析必須
