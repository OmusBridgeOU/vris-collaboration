# 札幌VketちゃんフォトスポットAR @ STYLY

<img src="/lineup/photospot.png" width="300px"></img>

## 企画概要
STYLYでさっぽろテレビ塔をスキャンすることで、本イベント限定のARコンテンツをお楽しみいただけます。

## 操作手順
1. STYLYアプリをスマートフォンにダウンロード
  - [iOS版](https://apps.apple.com/jp/app/styly/id1477168256)
  - [Android版](https://play.google.com/store/apps/details?id=com.psychicvrlab.stylymr&hl=ja&pli=1)

2. [VketSapporoChan PhotoARシーン](https://gallery.styly.cc/scene/7994bb4d-eefb-4010-a6f9-8f0e91dce59f)にアクセス

3. 起動ボタンをクリック。STYLYアプリを起動し、シーンを読み込む。

4. 札幌テレビ塔をスキャンして、ARコンテンツを楽しむ！
