# VketReal in 札幌 2025 summer 広報協力メンバー向け募集要項

# 主な活動内容と期間

広報協力メンバーに就任された方には、以下の活動をお願いします。

【活動期間】

就任期間：任命～2025年7月21日

【主な活動内容】

* XやYouTubeなど、自身の媒体でのVketReal in 札幌のPR、来場促進活動。  
* 公式ハッシュタグ（\#VketRealSapporo）を付けた投稿。  
* 「VketReal in 札幌」公式Xアカウントの投稿のリポスト・引用リポスト。  
* 会場に来場できる方は、イベント当日の会場・出展ブースの様子などを、写真や動画で発信。

※活動頻度の目安として、活動期間中は週に2〜3回程度の継続的な発信をお願いします。  
また、活動期間終了後も関連イベントなどでお願いする可能性があります。

# 広報協力メンバー特典

広報協力メンバーとして参加していただける方は、クレジットを公式サイトに掲載させていただきます。（広報協力による金銭的な報酬は発生いたしません。）

# 必須事項

* X、YouTubeなど発信・拡散が可能なSNSを運用中の方。  
* 誹謗中傷やネガティブな言動をせず、ポジティブに活動に取り組んでいただける方。  
* ルールを守り、自発的に活動できる方。  
* 協調性があり、お互いをリスペクトしあえる方。  
* VketReal in 札幌、ひいては北海道のメタバース・VR業界を盛り上げたい方。  
* VketReal in 札幌 の公式サイトやXなどで掲載可能なアバターをお持ちの方。  
* アバターが公序良俗に反するようなものでない方。

# 禁止事項

* 公序良俗に反する活動や、法令に違反する行為。  
* 暴力的、過激な発言、他者を落としたり誹謗中傷する事。  
* 個人的な人脈形成のみを目的とした活動（ネットワーキングなど）。  
* 特定の宗教やビジネス、団体への勧誘行為。  
* 強引に来場や特定の行動を促すこと。  
* 自身の行為に対して過度な見返りや感謝を求める言動。  
* 「VketReal in 札幌」の公式関係者（実行委員等）であるかのように振る舞う行為。  
* 実行委員会が許可していない情報（内部情報など）や、ロゴ・キャラクター等を無断で利用すること。

この他、実行委員会のルールに従い活動が行えない場合は任命後でも即その任を解除いたします。
